# 📡 影片掛網進度追蹤工具

追蹤「驗檔完成 → 掛網完成」的工作進度，自動標示延誤等級，WD 編號逾期即時警示。

## 怎麼看結果

- **線上版（手機也能看）**：https://xinrong-rgb.github.io/video-upload-tracker/
- 本機版：直接用瀏覽器打開本資料夾的 **index.html**

每天早上 9:30 排程自動更新並推送 GitHub；想手動更新就跟 Claude 說「更新掛網追蹤」。
（GitHub repo：https://github.com/xinrong-rgb/video-upload-tracker，公開倉庫）

## 運作流程

1. **每天 09:30**（排程 `video-upload-tracker`，需 Claude Code 應用程式開著；沒開的話下次啟動會補跑）
2. 掃描 admin@hwadzan.com 裡 amtbwork@hwadzan.org 寄來的「驗檔完成」「掛網完成」通知信
3. 解析 編號／標題／驗檔完成／掛網建立／掛網完成 時間，合併進 `data.json`
4. 執行 `build_dashboard.py` 重建 `index.html`
5. **WD 編號超過 2 個工作天（不含六日）未掛網 → 跳 macOS 通知（含警示音）**
6. 在 `log.md` 追加執行紀錄

## 延誤分級（工作天，六日不計）

| 工作天 | 顏色 | 意義 |
|---|---|---|
| ≤2 天 | 🟢 綠 | 正常 |
| 3–4 天 | 🟡 黃 | 注意 |
| 5–6 天 | 🟠 橘 | 偏慢 |
| ≥7 天 | 🔴 紅 | 嚴重延誤 |

## 檔案說明

| 檔案 | 用途 |
|---|---|
| `index.html` | 儀表板（瀏覽器開啟） |
| `data.json` | 所有項目的進度資料（以編號為 key） |
| `build_dashboard.py` | 由 data.json 重建儀表板＋輸出逾期摘要 |
| `log.md` | 每日執行紀錄 |
